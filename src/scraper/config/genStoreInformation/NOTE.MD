- ¿Por qué uso IA para esto?

Si haz explorado el proyecto lo suficiente, notarás que existe una branch llamada ["Classes-extracting"](https://github.com/AikePetunia/AAS-backend/tree/classes-extracting), donde exploré buscar una manera mucho mas fácil de generar las configuraciones necesarias para cada pagina web, pues, por cada pagina, tiene que tener info, categorias y lo más dificil, los **selectores**.
En esa branch, intenté hacer algo bastante similar a lo que hice con path extracting, pero no funciono pues cada página es un mundo y no es algo viable.
Por lo tanto, para extraer los selectores de la manera más rápidamente y sin errores, se usará ias agenticas para la tarea.
